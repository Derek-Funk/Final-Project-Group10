• PyTorch 3D-CNN similar to PyTorch MNIST example, with a few differences:

    • output to just 2 classes
    • use log softmax for last activation function
    • use binary cross entroy as criterion

  • Model 1:

    • start with just 10,000 training images
    • 71% test accuracy

  • Model 2:

    • switch from 3x3 kernel size to 2x2
    • 81% test accuracy

  • Model 3:

    • double training size to 20,000
    • 82% test accuracy

  • Model 4:

    • use all ~200,000 training images
    • 85% test accuracy
